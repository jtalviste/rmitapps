# Getting Started

# Server

In server folder

```sh
./gradlew bootRun
```

to start the server.

## Client

In client folder

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

You can click through the router links in order and
use the pre-entered data. Press the main button on each
page (in order) and you should see the results.
